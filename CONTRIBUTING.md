# Contributing

Thanks for helping improve Teul.

## Setup

```bash
pnpm install
pnpm dev
```

## Before you open a PR

```bash
pnpm format    # Prettier
pnpm lint      # ESLint
pnpm test:e2e  # Playwright
```

All three should pass.

## Pull requests

- Keep changes focused — one concern per PR.
- Write a clear title and explain the *why*, not just the *what*.
- Match the surrounding code style; don't customize shadcn primitives.

## Bugs and ideas

Open an issue. Include a minimal repro for bugs.
